## :ringed_planet: Star Wars Application

### :recycle: Available Scripts and Commands

```bash
# Install dependencies
npm i
yarn install
```

```bash
# Run app
npm run start
yarn start
```

### :hammer_and_wrench: Stack:

- ReactJS + Hooks / Custom Hooks
- React Router
- Context API
- Redux
- Axios
- Lodash
- PropTypes
- ClassNames
- SCSS / Mixins
